How it works

01

You enter your name

The form sends a GET /api/joke?name=… request to our Express.js server.

02

Express picks a category

The server hashes your name and selects a joke category — same name always picks the same type.

03

Axios fetches the joke

Axios makes a server-side HTTP call to JokeAPI v2 and returns the result as clean JSON.

04

You laugh (hopefully)

The client receives the JSON and renders the joke — no page reload needed.
